# Рейтинг ИИ-проводников 2026

Интерактивная страница рейтинга активности ИИ-проводников Ростелекома: таблица менторов, фильтры, бонусная программа, светлая и тёмная тема.

Дизайн и контент согласованы с лендингом [afisha-university.rt.ru/ai_mentor](https://afisha-university.rt.ru/ai_mentor).

## Быстрый старт

Страница работает без сборки и сервера — достаточно открыть `index.html` в браузере.

```bash
# или через локальный сервер (удобнее для разработки)
python3 -m http.server 8080
# → http://localhost:8080
```

## Структура проекта

| Файл / папка | Назначение |
|---|---|
| `index.html` | Страница рейтинга (HTML + CSS + встроенные JSON-данные) |
| `app.js` | Логика: таблица, фильтры, тема, поп-апы с бонусами |
| `fonts-embedded.css` | Rostelecom Basis Regular (base64) — для надёжной загрузки шрифта |
| `fonts/` | Остальные начертания Rostelecom Basis (Light, Medium, Bold) |
| `mentors-data.json` | Данные рейтинга (генерируется из Excel) |
| `build.py` | Обновление JSON и встраивание данных в `index.html` |
| `embed_fonts.py` | Пересборка `fonts-embedded.css` после замены шрифта |
| `ИИ-проводники_Статистика_2026.xlsx` | Исходная таблица с данными |

## Обновление данных из Excel

1. Замените или обновите файл `*2026.xlsx` в корне проекта.
2. Запустите:

```bash
python3 build.py
```

Скрипт обновит `mentors-data.json` и блок `<script id="mentors-data">` в `index.html`.

**Важно:** кварталы Q1 и Q2 — накопительные показатели, их нельзя суммировать. Уровни бонусной программы пересчитываются по правилам из `Ступени.pdf`.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub и загрузите содержимое папки:

```bash
git init
git add .
git commit -m "Рейтинг ИИ-проводников 2026"
git branch -M main
git remote add origin https://github.com/<username>/<repo>.git
git push -u origin main
```

2. В настройках репозитория: **Settings → Pages → Build and deployment → Source: GitHub Actions**.
3. После push в `main` workflow `.github/workflows/pages.yml` опубликует сайт автоматически.

Страница будет доступна по адресу `https://<username>.github.io/<repo>/`.

## Шрифты

В проекте используются корпоративные шрифты **Rostelecom Basis** из папки `fonts/`. Перед публикацией в открытый репозиторий убедитесь, что это допустимо политикой компании.

Если Regular-шрифт обновился:

```bash
python3 embed_fonts.py
```
